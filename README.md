# MindMap SOA System

## Опис
Цей репозиторій містить сервісно-орієнтовану архітектуру для роботи з ментальними картами.  
Система складається з трьох автономних сервісів, кожен з яких реалізує окрему функціональність.

---

## Сервіси

| Сервіс | Порт | Призначення |
|--------|------|------------|
| user-service | 8081 | Керування користувачами, автентифікація та авторизація |
| mindmap-service | 8082 | Збереження та обробка ментальних карт, нод та ребер |
| export-service | 8083 | Експорт ментальних карт у формати TXT, CSV, JSON |
